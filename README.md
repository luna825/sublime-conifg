
## 常用插件
#### babel 
支持ES6，React.js, jsx代码高亮，对 JavaScript, jQuery也有很好的扩展。
#### SideBarEnhancements：
SideBarEnhancements是一款很实用的右键菜单增强插件，对则边栏进行扩展,功能相当丰富。
#### Emmet 
前端HTML/CSS编写快速编写神器。
#### CSS3
支持所有CSS3语法提示
#### ReactJS 
让React组件内的HTML编写支持Emmet的写法
#### DocBlockr
注释编写快速生成
#### IMESupport 
解决中文输入法不跟随光标移动的问题
